# **🛡️ Cognitive Cyber-Threat Intelligence: Hybrid Neural Ensemble for Spam & Payload Filtering**

## **📋 Project Overview**
Modern enterprise communication channels are constantly targeted by sophisticated phishing, promotional spam, and malicious social engineering payloads. Traditional keyword-based filters or simple standalone machine learning models often fail when dealing with highly adaptive adversarial text patterns.

This project introduces an end-to-end Dual-Engine Deep Learning Hybrid Framework designed to intercept, analyze, and classify text streams with industrial-grade precision. The core architecture leverages two distinct artificial neural network paradigms working in a soft-voting consensus loop:

* Engine 1 (Deep Multi-Layer Perceptron Core): Focused on global statistical correlations, raw feature weight distribution, and behavioral patterns.

* Engine 2 (Advanced 1D Convolutional Neural Network Core): Focused on localized spatial feature scanning, identifying relationships between adjacent numerical markers (e.g., structural correlations between text lengths and link counts).

### The entire system is integrated into a unified pipeline—from raw natural language processing (NLP) regex metrics and data leakage safeguards to persistent HDF5 binary serialization—and deployed via a highly interactive live web frontend using the Gradio ecosystem.
